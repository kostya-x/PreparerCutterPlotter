Этот макрос нужен мне, что бы подготовить файл
для плоттерной порезки на плоттере GCC Jaguar V.
Файл подготавливается в программе CorelDraw X3.

--------------------------------------------------

Для началша нужна коррекция контура порезки на 0,3мм влево и 0,5мм вверх.

Удалить все направляющие.
(Иначе плоттер попытается их порезать. Иногда они скрыты)

Изменить толщину линии на "Волосяную линию".
(Плоттер режет только линии с такой толщиной)

Метки для плоттера должны быть залиты цветом R30 G24 B21
и у них не должно быть линии.
(Плоттер даже не начнёт резать, если это будет не так)

Сгруппировать контур для порезки и изменить цвет линий на чёрный.
(На это нет объективной причины :))
